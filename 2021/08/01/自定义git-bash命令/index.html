Created2021-08-01|Updated2021-12-30

目的

我希望在写笔记的时候，可以尽量少的切换软件和路径，如果可以通过一个命令行工具加markdown工具就可以完成编写，就十分不错，因此有了接下来的折腾。

准备

  1. 需要一台电脑(这里是win10系统)；
  2. 需要安装好git，typora;
  3. 我使用的是hexo管理文章，其他工具类似，自行探索；

实践

我是凑数的

1. 使用windos terminal

这东西在windows是相当好用的命令行工具，在其中添加git bash。可以在设置界面傻瓜式添加和设置git bash，不需要修改json文件。
